# ShopGrok Technical Challenge

### Challenge 1

Use Chrome Scraper and Xpath to scrape three websites.

Results are compiled in a pdf located in ```Scraper_challenge``` directory. Textual output is also provided.

### Challenge 2

Create a basic web crawler + use Python regex to extract data from string and make it json serialisable.

Scrapy application, CSV output as well as the Python files are located in ```Python_challenge``` directory.
